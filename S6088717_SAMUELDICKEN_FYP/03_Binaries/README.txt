Restore Git Repository

1: Extract Munchkin.git from the zip folder.
2: Open GitHub PowerShell. (You may need to install GitHub on your computer first).
3: Type: git clone file://
4: Drag the newly extracted folder into the powershell window
5: Type a \ at the end. 
6: Drag the folder where you want to store the new git repo into the powershell window.
7: Press the Enter Key
8: Powershell will begin cloning the git repo into a new repository.
9: Open GitHub Desktop and add a new repository
10: Browse to the repository you just created and click 'Add repository'

The entire git repo along with all branches and commit history will now be available to browse.
